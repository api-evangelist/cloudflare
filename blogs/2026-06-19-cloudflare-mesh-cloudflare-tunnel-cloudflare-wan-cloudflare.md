A unified Routes dashboard page consolidates route management across Cloudflare Mesh, Tunnel, WAN, and Magic Transit. Users can visualize networks with interactive maps showing equal-cost multi-path routing, manage routes in a single table, create, edit, and delete routes by connector name, and test routes before deployment.
