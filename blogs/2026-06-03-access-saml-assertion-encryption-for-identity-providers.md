Cloudflare Access now enables SAML assertion encryption, where identity providers encrypt authentication data using Cloudflare-managed certificates before transmission, protecting sensitive identity information from browser-based threats. The feature includes automatic certificate generation, rotation without downtime, and PEM export capabilities for flexible deployment across identity provider configurations.
