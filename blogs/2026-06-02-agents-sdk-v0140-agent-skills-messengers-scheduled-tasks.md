Agents SDK v0.14.0 introduces Agent Skills for on-demand capabilities, Telegram chat messengers, declarative scheduled tasks, and reasoning steps within Workflows. The release also significantly improves durable chat recovery to ensure conversations survive a mid-turn deploy or Durable Object eviction, with better recovery signals and stalled stream handling.
