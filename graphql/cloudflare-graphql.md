# Cloudflare GraphQL API

The Cloudflare GraphQL Analytics API provides flexible access to analytics data across Cloudflare products including HTTP requests, firewall events, and load balancing metrics. Developers can query specific datasets, filter by dimensions, and aggregate data for custom reporting.

**Endpoint:** https://api.cloudflare.com

**Documentation:** https://developers.cloudflare.com/analytics/graphql-api/

**References:**

- Documentation: https://developers.cloudflare.com/analytics/graphql-api/
